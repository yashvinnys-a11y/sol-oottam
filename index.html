How to Play

4 Levels of Tamil Learning

  1. Runner 🏃: Catch the highlighted Tamil letter shown in the top bar. Avoid obstacles and collect all 10 words.
  2. Guided Tracing ✍️: Trace each letter over its guide. The check considers path length, on-guide accuracy and guide coverage. It is practice rather than a formal handwriting assessment.
  3. Sentence Fill 🧩: Choose the correct Tamil word to complete each sentence. Incorrect answers use one life.
  4. Read Aloud Practice 🗣️: Listen, read the sentence aloud, and use Tamil browser speech checking where supported. A clearly labelled self-confirmation fallback is provided when recognition is unavailable or unreliable.
  5. Progress and evaluation events are stored locally. Use Evaluation Data to export CSV or JSON for testing.
  6. Controls: Arrow Up/Down, W/S, on-screen arrows, or swipe up/down.
